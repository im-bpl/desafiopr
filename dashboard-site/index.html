Sobre o Dashboard

O Dashboard Desafio PR é uma aplicação interativa para visualização e análise de dados dos NREs (Núcleos Regionais de Educação), escolas e professores do Paraná. Esta aplicação permite explorar os dados de forma hierárquica, filtrar informações específicas, atualizar os dados semanalmente e exportar informações para análises externas.

Desenvolvido para facilitar o acompanhamento do desempenho educacional, o dashboard oferece uma interface intuitiva e responsiva, adaptada para uso em diferentes dispositivos.

Principais Funcionalidades

  • Visualização de métricas gerais (NREs, escolas, alunos, professores)
  • Gráficos interativos de desempenho
  • Filtros por NRE, escola e professor
  • Navegação hierárquica pelos dados
  • Interface responsiva para dispositivos móveis
  • Exportação de dados em diferentes formatos (Excel, CSV)
  • Exportação de gráficos como imagens (PNG)
  • Exportação do dashboard completo como PDF
  • Atualização de dados via upload de planilhas
  • Histórico de atualizações com métricas comparativas

Como Utilizar

Para acessar todas as funcionalidades do Dashboard Desafio PR, clique no botão "Acessar Dashboard" no topo da página. Você será redirecionado para a aplicação completa, onde poderá:

  1. Visualizar os indicadores gerais de desempenho
  2. Filtrar os dados por NRE e escola
  3. Explorar os gráficos interativos
  4. Exportar dados e visualizações
  5. Atualizar os dados através do upload de planilhas

Para mais informações sobre como utilizar o dashboard, consulte o Manual do Usuário.
